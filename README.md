# dzencode

## 📦 Установка зависимостей

Установите все необходимые зависимости:

```bash
npm install
```

---

## 🚀 Запуск проекта

### Запуск всего приложения (frontend + backend + WebSocket)

```bash
npm start
```

Запускаются параллельно:

* Vite (frontend)
* Express сервер
* WebSocket сервер

---

### 🔧 Раздельный запуск (опционально)

Frontend (Vite):

```bash
npm run dev
```

Backend (Express):

```bash
npm run server
```

WebSocket сервер:

```bash
npm run ws
```

---

## 🏗 Сборка проекта

```bash
npm run build
```

## 👀 Предпросмотр сборки
